가장 긴 증가하는 수열

```python
import sys
input = sys.stdin.readline

N = int(input())
A = list(map(int, input().split()))
dp = [0] * N
for i in range(N) :
    for k in range(i) :
        if A[i] > A[k] :
            dp[i] = max(dp[i],dp[k]+1) # 전에 있는 수열중 가장 길이가 큰거를 하나 더한다.
           
print(max(dp))
```
